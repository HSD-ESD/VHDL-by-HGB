VUnit by HGB

One of the developers of VHDLbyHGB, Jakob Jungreuthmayer, also developed the extension VUnit by HGB, which is an Visual Studio Code extension for VUnit.

With this extension you can run your VUnit tests within the sidebar of Visual Studio Code, as long as you have installed VUnit.
If you have not installed VUnit or know about it, you can learn more unter the feature VUnit or on the official Website of VUnit, which you can find here.

You can read about the extensions features here or directly go to the repository and check it out

Open Repository

VUnit by HGB is a fork of the "Vunit Test Explorer" by Henrik Bohlin, who stopped maintaining said repostitory.

HDLRegression by HGB

Jakob Jungreuthmayer, also developes and maintaines the extension HDLRegression by HGB for Visual Studio Code.

This Extension uses HDLRegression, which is an alternative to VUnit. And just like VUnit, you can run your HDL Design test cases.
If you have not installed HDLRegression or know about it, you can learn more unter the feature HDLRegression or on the repository of HDLRegression, which you can find here.

You can read about the extensions features here or directly go to the repository and check it out
