Installation

VHDLbyHGB can be installed in Visual Studio Code as an extension.

Requirements

Before installing VHDLbyHGB, you need to install Visual Studio Code, which you can download here.

Installation from Visual Studio Code Marketplace

To access the marketplace for Visual Studio Code follow these steps:

  1. Open Visual Studio Code.
  2. CLick on the Extension icon on the left hand side window or press Ctrl + Shift + X for Windows/Linux and Cmd + Shift + X for MacOS to open the Extensions view.
  3. In the search bar at the top of the Extensions view, type in the extension name VHDLbyHGB or browse for the extension by clicking the "Explore" or "Extensions" tabs.
  4. Once you see the Extionsion "VHDLbyHGB" click on the "Install" button next to the name.
  5. Once the Extension is installed, you may need to reload Visual Studio Code. If so, you will see a "Reload" button next to the extension name.
  6. If Visual Studio Code has reloaded, the extension should be available for use.

Supported Tools

Within our extension you, can use VUnit, HDLRegression, and Quartus if they are installed on your device.
If you do not have VUnit installed, you can check it out here.
If you do not have HDLRegression installed, you can check it out here.
If you do not have Quartus installed, you can check it out here.
