VHDLbyHGB for Visual Studio Code Documentation

VHDLbyHGB is an open-source IDE for FPGA and ASIC developers. The IDE's goal is to provide simple and powerful software for professional HDL developers and students. Some tools come from open-source projects, while others have been developed by a group of students at Univertity of Applied Science Upper Austria - Campus Hagenberg.

If you find any bugs or have any questions, please open an issue on GitHub.
